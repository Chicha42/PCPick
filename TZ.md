# Техническое задание на проект: PCPick

## 1. Цель проекта
Создание сервиса для подбора оптимальной конфигурации ПК по заданному бюджету с учётом совместимости ключевых компонентов (CPU, GPU, MB, RAM). Сервис помогает пользователям получить готовую сборку, видеть актуальные рыночные цены и оценивать производительность через систему бенчмарков

## 2. Роли пользователей
- **Гость**: Вводит бюджет, получает варианты сборок, просматривает характеристики компонентов
- **Зарегистрированный пользователь**: Сохраняет сборки в личном кабинете, отслеживает динамику цен на графиках
- **Администратор**: Управляет базой через админ-панель, запускает обновление данных и контролирует актуальность бенчмарков

## 3. Модели данных
1. **CPU (Процессор)**  
   *Поля*: `name`, `socket`, `benchmark_score` (число), `price`, `url`
2. **GPU (Видеокарта)**  
   *Поля*: `name`, `pcie_version`, `benchmark_score`, `price`, `url`
3. **RAM (Оперативная память)**  
   *Поля*: `name`, `ram_type`, `ram_bar_count`, `price`, `url`
4. **Motherboard (Материнская плата)**  
   *Поля*: `name`, `socket`, `ram_type`, `pcie_version`, `ram_slots`, `price`, `url`
5. **Build (Сборка)**  
   *Поля*: `user` (связь ForeignKey), `cpu` (связь ForeignKey), `gpu` (связь ForeignKey), `mb` (связь ForeignKey), `ram` (связь ForeignKey), `total_price`, `created_at`
6. **PriceHistory (История цен)**  
   *Поля*: `component_id`, `component_type`, `price`, `date_checked`

## 4. Ключевой функционал
* **Автоматизированный подбор**: Алгоритм ищет лучшие компоненты по `benchmark_score`, входящие в бюджет и соответствующие правилам совместимости:
  - CPU и Motherboard: совпадение по полю `socket`
  - RAM и Motherboard: совпадение по полю `ram_type` и проверка, чтобы `ram_bar_count` не превышало `ram_slots`
  - GPU и Motherboard: проверка поддержки `pcie_version`
* **Визуализация цен**: На основе модели `PriceHistory` строятся графики изменения стоимости компонентов
* **Личный кабинет**: Сохранение и просмотр ранее созданных конфигураций

## 5. Внешние интеграции и аналитика
* **Внешнее API**: Получение актуальных показателей производительности (Benchmark scores) из внешних источников (GitHub репозитории или открытые датасеты)
* **Парсинг**: Сбор актуальных цен и названий комплектующих с доступных площадок ритейлеров электроники
* **Визуализация (Matplotlib)**: Генерация графиков истории цен для наглядного анализа выгодности покупки в данный момент
